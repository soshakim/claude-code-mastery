# 개발자 웹 이력서 개발 로드맵

## 프로젝트 개요
개발자 포트폴리오용 웹 이력서 사이트를 개발합니다.
HTML, CSS, JavaScript, Tailwind CSS를 활용하여 모던하고 반응형 웹사이트를 구축합니다.

---

## Phase 1: 프로젝트 설정 및 구조 구성
**목표:** 프로젝트의 기초 구조 완성

### Tasks
- [ ] 프로젝트 디렉토리 구조 생성
  - `index.html` - 메인 페이지
  - `css/` - 스타일 파일 디렉토리
  - `js/` - JavaScript 파일 디렉토리
  - `assets/` - 이미지 및 리소스 디렉토리

- [ ] HTML 기본 템플릿 작성
  - Tailwind CSS CDN 링크 추가
  - 메타 태그 설정 (반응형 웹 설정)
  - 기본 구조 레이아웃

- [ ] Tailwind CSS 설정
  - CDN을 통한 기본 설정 또는 빌드 프로세스 구축

---

## Phase 2: 헤더 및 네비게이션 구현
**목표:** 웹사이트 최상단 헤더 및 네비게이션 완성

### Tasks
- [ ] 헤더 디자인
  - 로고/이름 표시
  - 네비게이션 메뉴 (Home, About, Projects, Contact)
  - 반응형 모바일 메뉴 (햄버거 메뉴)

- [ ] 네비게이션 스타일링
  - Tailwind CSS를 이용한 디자인
  - 호버 효과 추가

- [ ] JavaScript 기능 구현
  - 모바일 메뉴 토글 기능

---

## Phase 3: 메인 콘텐츠 영역 구현
**목표:** 이력서 주요 섹션 완성

### 3.1 Hero 섹션 (소개)
- [ ] 프로필 이미지/아바타
- [ ] 인사말 텍스트
- [ ] CTA 버튼 (이력서 다운로드, 연락하기)

### 3.2 About 섹션 (소개)
- [ ] 자기소개 텍스트
- [ ] 기술 스택 목록
  - 프론트엔드 기술
  - 백엔드 기술
  - 기타 기술

### 3.3 Experience 섹션 (경력)
- [ ] 경력 항목 카드 디자인
- [ ] 회사명, 직책, 기간, 설명
- [ ] 타임라인 스타일 레이아웃

### 3.4 Projects 섹션 (프로젝트)
- [ ] 프로젝트 카드 그리드 레이아웃
- [ ] 프로젝트 이미지
- [ ] 프로젝트 설명, 사용 기술, 링크
- [ ] 호버 효과 및 상호작용

### 3.5 Skills 섹션 (스킬)
- [ ] 스킬 카테고리별 정렬
- [ ] 프로그레시브 바 또는 배지 형태 표시
- [ ] 숙련도 시각화

## Phase 5: 상호작용 및 애니메이션
**목표:** 동적 효과 추가로 사용자 경험 향상

### Tasks
- [ ] 스크롤 애니메이션
  - Fade-in 효과

- [ ] 버튼 호버 효과

- [ ] 부드러운 스크롤 네비게이션

  - 스크롤 시 활성 메뉴 하이라이트

  - 라이트/다크 테마 전환

## Phase 6: 성능 최적화 및 반응형 디자인
**목표:** 모든 디바이스에서 완벽하게 작동하는 웹사이트

### Tasks
- [ ] 반응형 디자인 검증
  - 모바일 (< 640px)
  - 태블릿 (640px - 1024px)
  - 데스크톱 (> 1024px)

- [ ] 이미지 최적화
  - 압축
  - 적절한 크기 설정
  - 레이지 로딩 (필요시)

- [ ] 페이지 로드 속도 최적화
  - CSS 최소화
  - JavaScript 최소화
  - 불필요한 파일 제거

- [ ] SEO 최적화
  - 메타 디스크립션
  - 오픈 그래프 태그
  - Structured Data (JSON-LD)
